# F1 Race Predictor: 2026 Australian GP

This project uses Machine Learning and historical Formula 1 data to predict the finishing order of the 2026 Australian Grand Prix. By using the `fastf1` library to extract lap times, telemetry, and past results, the model analyzes both race pace and one-lap performance to generate a realistic final race classification.

## Project Overview

This project goes beyond simply looking at the starting grid by calculating advanced metrics like **tyre degradation** and **median race pace** during Free Practice 2 (FP2). It combines these with Qualifying results and a custom team performance score (weighted for the new 2026 regulations) to predict the final race standings. Since the prediction is for the first race of 2026, there is no data about previous races with these new regulation.

The core of the project is an **XGBoost Regressor** model, trained on historical data from the 2024 and 2025 championships.

## The project's steps

The project is broken down into several logical steps:

* **Data Acquisition (2026):** Extracting Qualifying and FP2 results for the Australian GP using the `fastf1` API.
* **Feature Engineering:**
* `Quali_Delta_Pct`: The percentage gap between a driver's qualifying time and the Pole Position time.
* `FP2_Delta_Pct`: The percentage gap between a driver's median race pace and the best FP2 race pace.
* `Tyre_Degradation`: Calculated by applying a linear regression to the lap times of long runs during FP2.
* `TeamPerformanceScore`: A baseline score calculated from 2025 championship points, multiplied by a custom modifier (`RegChangeBoost`) to simulate the impact of the new 2026 technical regulations.


* **Historical Training:** The script dynamically builds a training dataset by extracting these same metrics for all non-sprint races from the 2024 and 2025 seasons.
* **Modeling:** Missing data is handled using a `SimpleImputer` (median strategy), and the final finishing positions are real numbers predicted by an `XGBRegressor` model.

## 🛠️ Technologies Used

* **Python 3.x**
* **FastF1:** For accessing official Formula 1 timing and telemetry data.
* **Pandas & NumPy:** For DataFrame manipulation and mathematical/statistical calculations (e.g., using `np.polyfit` for tire degradation).
* **Scikit-Learn:** For data preprocessing (`SimpleImputer`).
* **XGBoost:** The gradient boosting algorithm used for the regression model.

## 🚀 Installation and Usage

To run the notebook or script locally, make sure you have the necessary libraries installed.

1. Clone the repository.
2. Install the required packages via terminal:
```bash
pip install pandas numpy fastf1 xgboost scikit-learn

```


3. Run the script. *Note: The `fastf1` library will cache data in a local folder named `cache` to significantly speed up future executions.*

## Expected Output

The script outputs a sorted DataFrame containing the race predictions:

* Final Predicted Position
* Driver Abbreviation (e.g., VER, LEC, NOR)
* Team Name
* Raw Predicted Race Position (the exact float value generated by the regressor)
